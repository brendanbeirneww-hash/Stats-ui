# Stats UI Deployment Guide

## How to Test Locally
1. Extract the ZIP file.
2. Open `index.html` in any browser.
3. Resize the browser or use Developer Tools (Ctrl+Shift+M) to simulate mobile.

## How to Deploy Online
### GitHub Pages
1. Create a GitHub repository.
2. Upload `index.html`.
3. Go to Settings > Pages > Select branch and root.
4. Access your site at `https://username.github.io/repo-name/`.

### Netlify
1. Go to https://netlify.com.
2. Drag and drop the folder.
3. Get your live URL instantly.

### Vercel
1. Go to https://vercel.com.
2. Upload or connect GitHub repo.
3. Deploy and get live link.

Enjoy your responsive Stats UI!
